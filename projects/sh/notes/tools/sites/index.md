# Work Sites Dispatch

Keyboard-driven development hub for navigating project sites.

## Keyboard Shortcuts

### Projects
- **W** - ws
- **D** - di
- **S** - sh
- **E** - en

### Modes
- **A** - app
- **X** - docs

### Actions
- **L** - local
- **T** - netlify
- **P** - public
- **R** - repo
- **Y** - deploy
- **N** - dns
- **B** - bubble

### Execute
- **Enter** - visit URL shown in feedback
- **Delete** - restart servers
