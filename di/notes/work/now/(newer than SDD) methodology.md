# Five things

Five things together are what make something an **intent**: 

1. description of what is wanted
2. the constraints around it
3. the scenarios under which it fails
4. the scenarios under which it succeeds
5. the connections
    1. the links to the other **intents** it touches
    2. so a change here is traceable to everything it affects
